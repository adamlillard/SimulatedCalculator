Sagacious Media Calculator Help Page 

Basic Button Functions

This section goes over the functions and uses of some of the calculators more complicated buttons

Clear Button

This button clears all text in the input field of the calculator. It can also be access with its short cut "C".

Reset Button

This button clears all text in the input field and  display field of the calculator. It can also be access with its short cut "R".

Plus Minus Button

This button changes the sign of a number in the input field to its inverese. If you pressed this button while having 65 in the input field it would change the number to -65 and vice versa. It can also be access with its short cut the shift button.

Inverse Button

This button changes a number in the input field to its inverse. If you pressed this button while having 2in the input field it would chane the number to   0.5 (1/2).

History Button

This button opens the history panel and shows all the past calculations done by the calculator. These calculations can also be printed in the menu under file. (file>print)

Power Of Button  

This button will calculate the power of problems. It can be used like any other button by clicking it after entering the basein the input field, then entering expontent and clicking the equals button.           


How to Use Units

 

When using units you must assign units to both values or the calculator will drop the units and calculate as though there are none. The drop down on the upper purple box controls how the units will be displayed at the end of the calculation.
